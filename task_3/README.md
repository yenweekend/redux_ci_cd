# React Redux Posts App

## Overview

This is a React + Redux Toolkit application that:

- Fetches posts from API
- Displays posts
- Allows adding a new post

---

## Tech Stack

- React
- Redux Toolkit
- TypeScript
- Vite

---

## API

https://jsonplaceholder.typicode.com/posts

---

## How to run

```bash
npm install
npm run dev
```
